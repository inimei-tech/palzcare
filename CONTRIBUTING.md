# Contributing to PalzCare

PalzCare ist aktuell ein Soloprojekt in der Validierungsphase. Developer-Recruiting ist aktiv.

## Interesse mitzumachen?

Schreib an: **palzcare@proton.me**

Bitte kurz schildern:
- Was du technisch kannst (Stack, Erfahrung)
- Warum dich das Projekt interessiert
- Wie viel Zeit du investieren könntest

## Geplante Tech-Bedürfnisse

- React Frontend (Mobile-first)
- Node.js Backend
- Supabase / PostgreSQL
- DSGVO-konforme Infrastruktur
- Matching-Algorithmus
- Push Notifications

## Prinzipien

- Privacy by Design — keine Features ohne DSGVO-Prüfung
- Mobile-first
- Einfachheit vor Komplexität (Zielgruppe: Mütter & Senior:innen)
- Kein Over-Engineering in der Frühphase
